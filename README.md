# siteblocker
A site blocker extension using The "-site:XXX.com" mechanism in Google search.

## How to use:

Just toggle the developer mode of the extension part of your google chrome on, then add all files of the extention to it.

Then just follow what The GUI said. You can block some sites you do not want to see.

**Note**: Google AI mode may be disabled with `site:` tag in search bar.

Thanks and enjoy.
